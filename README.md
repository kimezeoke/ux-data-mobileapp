# ux-data-mobileapp
UX and data systems project analyzing campus parking inefficiencies using student research, survey data, and system design. Built and tested an interactive Figma prototype validated by 90% of users for improved usability and intent to use.

# Campus Parking UX Project

## Overview
User research and UX/data systems project analyzing campus parking inefficiencies using student survey data.

## Tools Used
Figma, Google Forms, Excel

## Key Work
- Conducted user research with 20+ students  
- Analyzed survey data to define product requirements  
- Built and tested interactive Figma prototype  

## Outcome
90% of users reported improved usability and intent to use the app
